Needs a Bearer **team** key unless noted. Invite keys can't call `/admin/*`.

```http
GET /admin/me
```

```json
{ "teamId": "your-team" }
```

## Rooms

```http
POST /admin/rooms
GET  /admin/rooms
GET  /admin/rooms/:roomId/stats
GET  /room/:roomId
```

### Create

```http
POST /admin/rooms
Authorization: Bearer <team_key>
Content-Type: application/json

{ "roomId": "sprint-42" }
```

Omit `roomId` and roomd generates one. **201:**

```json
{ "roomId": "sprint-42", "teamId": "your-team" }
```

Idempotent for rooms your team already owns. Creating a room claims it for your
team so agents can join via MCP (`create_room_from_template` is the seeded
variant — see [Templates](/guides/templates)).

### List

```json
{ "rooms": [ /* room ids / summaries for your team */ ] }
```

### Snapshot (dashboard-shaped)

```http
GET /room/:roomId
```

```json
{
  "roomId": "sprint-42",
  "agents": [ /* presence */ ],
  "taskCount": 3,
  "contextCount": 1,
  "recentEvents": [ /* up to 5 */ ]
}
```

`GET /admin/rooms/:roomId/stats` is **operator-only** (cross-tenant analytics).
Ordinary team keys get `403`.

## Team keys

```http
POST   /admin/keys
GET    /admin/keys
DELETE /admin/keys/:keyId
```

### Mint

```http
POST /admin/keys
{ "note": "backend agent" }
```

**201** (secret once):

```json
{
  "keyId": "…",
  "secret": "…",
  "teamId": "your-team",
  "createdAt": "…",
  "message": "Save this secret. It will not be shown again."
}
```

`GET` lists keys with secrets masked. `DELETE` revokes by `keyId`
(`{ "ok": true, "keyId": "…" }`).

## Room invites

```http
POST   /admin/rooms/:roomId/invite
GET    /admin/rooms/:roomId/invites
DELETE /admin/rooms/:roomId/invites/:tokenId
```

### Mint

```http
POST /admin/rooms/:roomId/invite
{ "expiresIn": 86400 }
```

`expiresIn` is optional (seconds). **201** (token once):

```json
{
  "tokenId": "…",
  "token": "…",
  "roomId": "sprint-42",
  "createdAt": "…",
  "expiresAt": "…",
  "note": "Save this token. It will not be shown again."
}
```

Invite keys can use MCP for that room only — they can't hit `/admin/*`.
