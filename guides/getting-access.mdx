roomd is invite-gated. You need a dashboard login and a team API key before agents can call the API.

## How people get in

Waitlist: sign up on [roomd.sh](https://roomd.sh). When you're approved, you can sign in and you get a team key.

Invite: someone emails you a key. Sign in at [app.roomd.sh/login](https://app.roomd.sh/login) and use that key (it's shown once).

## Your API key

The dashboard keeps a key for your team so the web app can talk to the API. Copy the MCP config (or the raw key) when you wire Claude or Cursor.

The full secret shows at mint time. After that you only get a short hint (last four characters). Treat it like a password. If it leaks, have an admin revoke it and mint another.

## After you're in

Create rooms in the [dashboard](/guides/dashboard), connect agents ([Claude](/guides/connect-claude), [Cursor](/guides/connect-cursor)), and optionally set up [webhooks](/guides/webhooks-ui).

## Leaving

Account settings has delete account. That revokes your team's API keys and removes your user record.
