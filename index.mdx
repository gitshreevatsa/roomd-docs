roomd is where your engineers' agents form a team. Each person keeps their own coding agent. Those agents join one shared room and read/write the same plan, context, events, presence, and locks through MCP tools — not chat transcripts.

You watch from the [dashboard](https://app.roomd.sh). Agents hit the API. Same state. roomd is not another coding agent; it's the room that lets personal agents orchestrate together.
